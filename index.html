About Silent Speak

Silent Speak enables users to convey messages through Morse code, offering a voice to those who may not have one in traditional communication settings.

At the heart of Silent Speak lies a sophisticated hardware setup, integrating touch sensors capable of detecting both dots and dashes, the fundamental elements of Morse code. These sensors, meticulously crafted and calibrated, provide users with a tactile interface to input their messages effortlessly. The signals from the touch sensors are then swiftly processed by an Arduino microcontroller, which translates them into meaningful characters, ready to be displayed.

On the software front, Silent Speak boasts a versatile application developed using Flutter, ensuring seamless compatibility across various devices and platforms. This user-friendly app serves as the bridge between the hardware and the user, facilitating intuitive interaction and efficient message transmission. Additionally, a dedicated website built with ReactJS offers further accessibility, enabling users to access Silent Speak’s functionalities from any web-enabled device.

Central to Silent Speak’s mission is its commitment to accessibility and inclusivity. In situations where verbal communication is not an option, Silent Speak provides a lifeline through Morse code. One remarkable feature of Silent Speak is its emergency alert system, activated by a specific Morse code sequence. By tapping two dots followed by two dashes, users can trigger an alarm, signaling a call for help to those around them.

Silent Speak represents a paradigm shift in communication technology, transcending barriers and empowering individuals to express themselves in ways previously unimaginable. Whether it's aiding those with speech impairments, facilitating communication in noise-sensitive environments, or serving as a backup communication tool in emergencies, Silent Speak stands at the forefront of innovation, transforming lives one tap at a time.


Technology Used in the Project


Software

Flutter Development




Morse Code Converter and Games App Description: Silent Speak is a versatile mobile application developed using Flutter. Its primary functionality is to convert text input from 28 languages into Morse code. This feature is particularly useful for individuals who cannot speak and need an alternative means of communication.

Additionally, the app includes interactive games such as Tic Tac Toe, Rock Paper Scissors, and Flip Card, providing entertainment and engagement for users.

Hardware

Arduino

Silent Speak's hardware setup forms the foundation of its innovative communication system. Crafted with precision and ingenuity, the hardware seamlessly integrates touch sensors designed to detect both dots and dashes, the fundamental units of Morse code. These touch sensors provide users with an intuitive and tactile interface, enabling them to input messages effortlessly.
The signals from the touch sensors are swiftly processed by an Arduino microcontroller, a key component that serves as the brain of the system. Through meticulous calibration and programming, the Arduino translates the touch inputs into meaningful Morse code characters, which are then ready to be displayed.
Whether it's aiding individuals with speech impairments or facilitating communication in noise-sensitive environments, Silent Speak's hardware provides a robust platform for empowering communication in diverse scenarios.

Coding

Flutter , C++ and Python

The coding aspect of the Silent Speak project involves a sophisticated integration of Flutter, C++, and Python, each playing a crucial role in different facets of the system's functionality. Flutter, a versatile and powerful framework for building cross-platform applications, forms the backbone of Silent Speak's user interface. Through Flutter, developers create a seamless and intuitive application that serves as the primary interface for users to interact with the Silent Speak hardware. This includes designing the layout, implementing touch input functionality, and displaying the translated Morse code messages in real-time. On the hardware side, C++ is utilized for programming the Arduino microcontroller. C++ offers the flexibility and performance needed to efficiently process the signals from the touch sensors and translate them into Morse code characters. Through C++, developers implement the logic for detecting and interpreting touch inputs, ensuring accuracy and reliability in message transmission. Python plays a pivotal role in the backend of the Silent Speak system, handling tasks such as data processing, message storage, and integration with external services if needed.
